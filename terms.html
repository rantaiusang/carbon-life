Syarat dan Ketentuan

Dengan menggunakan aplikasi Carbon Life, Anda setuju untuk mematuhi ketentuan berikut:

1. Penggunaan Aplikasi

Aplikasi ini bertujuan memberikan informasi estimasi emisi karbon pribadi berdasarkan input harian Anda. Perhitungan bersifat perkiraan dan bukan pengganti audit lingkungan resmi.

2. Pembayaran Offset

Jika Anda memilih untuk melakukan offset karbon, pembayaran dilakukan menggunakan Pi Network dan tunduk pada ketentuan dari jaringan Pi itu sendiri.

3. Kewajiban Pengguna

  • Memberikan data yang akurat sesuai aktivitas harian Anda
  • Tidak menggunakan aplikasi untuk tujuan ilegal

4. Batasan Tanggung Jawab

Kami tidak bertanggung jawab atas kerugian atau kerusakan akibat penggunaan aplikasi ini. Aplikasi ini bersifat edukatif dan partisipatif.

5. Perubahan

Kami dapat mengubah syarat ini kapan saja. Versi terbaru akan tersedia di halaman ini.
